## Contributing

Contributions are what make the community such an amazing place to learn, inspire, and create. **Any contributions you make are greatly appreciated**.

To contribute:

1. Fork the [repository](https://github.com/2CodersStudio/awesome_2coders).  
2. Create a new branch:  
   ```bash
   git checkout -b feature/YourFeature
   ```  
3. Add your resource under the appropriate category with a short description.  
4. Commit your changes:  
   ```bash
   git commit -m 'Add a new awesome resource'
   ```  
5. Push to the branch:  
   ```bash
   git push origin feature/YourFeature
   ```  
6. Open a Pull Request and provide details about the resource.

**Guidelines**:

- **Quality Over Quantity**: We prefer a curated list of truly valuable links.  
- **Short Descriptions**: Keep descriptions concise.  
- **Check for Duplicates**: Don’t add an item if it already exists.  
- **Stay Awesome**: If it doesn’t inspire or help, it may not fit this list.
